# germ-slide

A research repository containing documents and reports related to germ/microbiology slide analysis.

## Repository Contents

| File | Description |
|------|-------------|
| `IEEE_Conf.pdf` | IEEE Conference paper related to the germ-slide research |
| `article_review.pdf` | Review of relevant articles in the field |
| `germ_report.pdf` | Detailed germ analysis report |

## Overview

This repository hosts academic and research documents for the **germ-slide** project, which focuses on the analysis and study of germ slides. The materials include conference proceedings, article reviews, and research reports.
